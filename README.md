# 🤖 Python学習習慣化システム (AIメンター・プロトタイプ版)

Gemini APIによる「課題生成・コードレビュー」と、GitHub APIによる「学習履歴の自動保存」を組み合わせた、Python学習習慣化のためのGoogle Colab用システムです。

## 🚀 主な機能
* **AI課題生成**: レベルや過去の履歴を考慮し、重複しない最適な難易度の課題を生成。
* **3段階ヒント**: 答えのコードは明かさず、実装の手がかりを段階的に提示。
* **AIコード診断**: 提出されたコードを採点し、アドバイスやリファクタリング案を提示。
* **GitHub自動同期**: 60点以上の合格コードをJSONログ化し、自動的にGitHubへコミット。

## 🛠️ 事前準備・セットアップ
1. Google Colabの「シークレット（鍵マーク）」に `GEMINI_API_KEY` と `GITHUB_TOKEN` を登録。
2. コード最上部の `GITHUB_USER` と `REPO_NAME` を自身の環境に合わせて書き換え。
3. 最初のセルを実行すると、自動的に環境チェックが行われます。

## 🤖 AIとの協働について
本プロジェクトの基盤コードは、生成AI（Gemini）と対話しながらプロトタイプを高速に構築しました。
プロンプトの工夫として、AIからの出力を確実にシステムでパースできるよう、レスポンス形式をJSON（`response_mime_type`）に固定する設計を指示・実装しています。
